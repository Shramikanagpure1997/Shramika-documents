# Conducting webinar 
Before 4-5 days of the upcoming batches, we create the links using your Zoom credentials.  
Login to Zoom using credentials.

1.  Once logged in, click on the "Meetings" tab.
 
 ![](https://i.gyazo.com/3af5ebb642efefe7e85c95f369fbafa4.png)
    
    
2.  Select the option to schedule a schedule meeting.

![](https://i.gyazo.com/d350522402cf905adf6aaa512b6e66f3.png)
    
3.  In the meeting settings, make the necessary changes like updating the topic name (e.g., "Java-oriented program"), setting the desired timings, and adjusting the duration.

![](https://i.gyazo.com/7e90767ee114b9ebc966481e42fdcd70.png)    
    
    
4.  Enable the cloud recording option.
    
![](https://i.gyazo.com/6d82751909f2c0d83387b79bbe5e0744.png)
    
5.  After making the changes, click on the "Save" option to save the meeting.

![](https://i.gyazo.com/bf788a88653e40a9a8f7c0fe63c655bc.png)
    
    
6.  Once the meeting is saved, copy the meeting link from the Zoom interface.    
![](https://i.gyazo.com/7376a4e045749af4d79393f6b99fe604.png)
    
7.  Open your Rebrandly credentials.
    
8.  Access the Rebrandly dashboard.

![](https://i.gyazo.com/dd69bf51d0c5eb9b199ad4470b5763b8.png)

9.  Choose the appropriate destination where you want to create a shortcut for the link (e.g., a specific channel).

![](https://i.gyazo.com/6166bdcba01011eaec4c6ef8d00c545c.png)
    
    
10.  If you will put curson on a specific link which you want to edit then you need to click on pencil icon  
    Save the link in Rebrandly and copy the shortened link provided by Rebrandly.
  ![](https://i.gyazo.com/b8197dcbaa1eaf75401ef3f5ea8c9e6c.png)
    
11.  You can now use the copied Rebrandly link to share it in the desired channel.
    

When it's time to start the session:

1.  Go to the Zoom credentials and log in.
    
2.  Click on the "Meetings" tab.

![](https://i.gyazo.com/a4ae8248eae95c422c59f95165928243.png)
    
3.  Find the scheduled meeting that is about to start within the next 10 minutes.

![](https://i.gyazo.com/606bf3ddf9ac3233a0e3415a6461be04.png)
    
4.  Click on the "Start" option to initiate the webinar or meeting.
 
 ![](https://i.gyazo.com/c278e4ac954c46eb4233d2cc3572865e.png)
